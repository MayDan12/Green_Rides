# GreenRide – Eco-Friendly Ride Booking

## Features

- View Electric/Hybrid rides
- Book ride with CO₂ impact
- Profile with EcoPoints & dark mode
- Map view with pickup/dropoff
- Persisted state (AsyncStorage via Zustand)

## Tech Stack

- React Native + TypeScript
- Expo
- Zustand (state)
- Styled Components
- React Navigation
- react-native-maps
- System dark mode

## Setup

````bash
git clone <repo>
cd GreenRide
npm install
npx expo start
````
